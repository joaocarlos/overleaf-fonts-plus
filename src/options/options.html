Overleaf Fonts+ Options

Control which custom editor fonts appear in Overleaf.

Choose which custom fonts are added to Overleaf's Menu → Settings → Appearance → Font family list. Only fonts that are installed on your system (or available from another source, like a font manager) will actually render in the editor; otherwise the browser will fall back to a default monospace font.
